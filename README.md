# Formula 1 Competitive Balance Analysis (1950–2025)

This project analyzes long-term competitive balance in Formula 1 using race-level
data from 1950 to 2025. The analysis focuses on constructor dominance, season-level
competitiveness, and the evolving importance of qualifying performance.

## Project Structure

- **Notebook 04 – Analysis Overview**
  - Exploratory analysis of all-time and era-based constructor dominance

- **Notebook 05 – Season-Level Competitiveness**
  - Quantitative metrics of dominance and competitive balance across seasons

- **Notebook 06 – Qualifying Importance Over Time**
  - Statistical analysis of the relationship between grid position and finishing
    position across different Formula 1 eras

## Key Outputs

- Era-based constructor dominance visualizations
- Season-level dominance metrics
- Long-term trends in qualifying importance

## Data

The analysis uses publicly available Formula 1 historical race data
covering seasons from 1950 to 2025.
